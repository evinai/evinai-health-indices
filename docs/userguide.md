### User Guide

### Usage
Using the short forms or abbreviated forms of indices
```python
>>> from health_indices import bmi,bai,
>>> bmi(54,1.70)

```

Using the long form of indices
```python
>>> from health_indices import bodymassindex
>>> bodymassindex(54,1.70)

```
